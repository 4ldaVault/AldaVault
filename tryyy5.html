Cara Post Akun

  1. Hubungi kontak Admin 3 (UNTUK POST AKUN).
  2. Kirim pesan setelah pesan otomatis dari admin: "MAU POST AKUN MIN!"
  3. Admin akan mengirimkan format untuk post akun.
  4. Penjual akan mengirimkan screenshot dan detail akun secara lengkap (spek, informasi akun, dan lainnya).
  5. Akun akan diposting di Status WhatsApp, Status Instagram, atau Postingan Instagram sesuai pilihan.
  6. Setelah pengisian format selesai, penjual akan memilih kategori post akun.
  7. Selanjutnya, penjual akan melakukan pembayaran untuk jasa posting.
